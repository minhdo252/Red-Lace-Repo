# NónAI — Official Brand Assets

These are the **finalized, official** logo and mascot for AITravelMate / NónAI.
One logo, one mascot — use these everywhere. Do not substitute variants.

## Logo (nón lá mark)
The mark is a stylised **nón lá** (Vietnamese conical hat) that doubles as a
**shield** — a nod to "Tourist Shield". The teal spark signals the AI layer.

| File | Use |
|------|-----|
| `logomark.svg` | Transparent mark — top bars, splash, inline on any background |
| `logo-badge.svg` | Mark on a sage badge — app icon look, favicons, marketing |
| `logo-maskable.svg` | Full-bleed safe-zone version — Android maskable icon source |

Rendered PWA icons live in `../icons/` (`icon-192`, `icon-512`, `icon-maskable`, `apple-icon`).

## Mascot ("Nón bé" — the nón lá girl)
A warm watercolour girl in áo dài wearing a nón lá, holding a lantern.
She is the friendly guide throughout the app.

| File | Use |
|------|-----|
| `mascot.png` | Full body — splash, onboarding, home greeting, empty states |
| `mascot-face.png` | Square face crop — avatars, chat, small greetings |
